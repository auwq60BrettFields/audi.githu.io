《是誰搞的鬼》（I Know What You Did Last Summer）是青春驚悚類型中最具代表性的系列之一，以“秘密、懲罰與報應”為核心，講述一群年輕人在夏天誤殺他人後，被神秘鐵鉤殺手盯上的懸疑故事。初代作品由珍妮佛·樂芙·休伊與小佛萊迪·普林茲主演，憑藉緊湊節奏與高能驚悚場面成為經典。
近年推出的續集與重啟版本延續了“我去年夏天做了什麼”的經典噱頭，同時加入原作角色回歸，讓新舊世代共同面對從過去陰影中復甦的恐懼，使故事更具延展性與時代感。
如今，《是誰搞的鬼》- 经典中文字幕 版本再次受到影迷關注，而 高清中文字幕 版本則提供更清晰的視聽體驗。此外，想要一次補完整個系列的觀眾，也能透過 中文字幕全集 形式輕鬆回顧整部青春驚悚傳奇。
